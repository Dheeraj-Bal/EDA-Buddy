## Making similar web-app

To create your own EDA web application, follow the following steps.

**Create a virtual environment named eda**

```bash
conda create -n eda python=3.11
```

Activate the virtual environment

```bash
conda activate eda
```

**Clone the repository**

```bash
git clone https://github.com/Dheeraj-Bal/EDA-Buddy.git
```



**Go to the project directory**


Install dependencies

```bash
  pip install -r requirements.txt
```
Make the changes 

**Launch the app**

```bash
streamlit run app.py
```
